* EitoKs/maze について *

このリポジトリはUnityで制作した作品のスクリプトをまとめた場所である。
なお、ここにあるすべてのソースプログラムは個人で作成した。
-----------------------------------------------------------------------
〇各ファイルについて
このリポジトリには計6個のファイルが存在している。
以下は各ファイルの大まかな説明である。

①FPSController
ゲーム中の移動に関することを記述したファイル。

②Goal
ゴール後の処理を記述したファイル。

③InputText
タイトル画面でステージの大きさを入力した値を処理するファイル。

④MazeStick
迷路をランダム生成するファイル。

⑤OptionScript
操作説明画面を管理するファイル。

⑥Sound
音の発生に関するファイル。

-------------------------------------------------------------------------------
〇フォルダ内のファイルの説明について
　各フォルダ内にあるファイルの説明はファイル内のコメント文にて示す。
